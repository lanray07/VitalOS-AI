# Humanized Asset Direction

App Store assets are generated from `AppStoreAssets/create_vitalos_assets.ps1`.
The generator composites exact rendered copy and app UI over the premium visual direction saved at `AppStoreAssets/References/premium-app-store-direction.png`.
It outputs both iPhone 6.5-inch screenshots and 13-inch iPad screenshots.

The current asset set uses a premium App Store poster style instead of abstract placeholders:

- morning recovery and sleep readiness
- adaptive daily protocol planning
- voice coach support with lifestyle imagery
- analytics and pattern review
- premium access and subscription review context

Keep imagery educational and wellness-focused. Do not imply diagnosis, treatment, emergency support, or medical monitoring.
